# Demo Walkthroughs

Collection of demo walkthrough resources organized by AWS service.  I use these walkthroughs and content for demos I do with my customers.  I use a HTML page that has instructions and goes over the service and individual components of the surface.  Github doesn't allow HTML rendering but you can download the HTML page and open within your browser.

## Structure

Each service has its own folder containing HTML walkthroughs and supporting files.

```
service-name/
  demo-walkthrough/
    walkthrough.html
```
