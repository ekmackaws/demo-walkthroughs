# Demo Walkthroughs

Collection of demo walkthrough resources organized by AWS service.  I use these walkthroughs and content for demos I do with my customers.  I use a markdown file that I follow along as well as any resourcecs I create in relation to the service and follow the flow of the markdown file during the demo.  Will continue to add more services as more demos happen.

## Structure

Each service has its own folder containing markdown file walkthroughs and supporting files.

```
service-name/
  demo-walkthrough/
    walkthrough.html
```
